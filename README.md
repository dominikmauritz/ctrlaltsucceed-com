# ctrlaltsucceed-com

Static copy of `https://ctrlaltsucceed.com`, rebuilt so it can be hosted directly on GitHub Pages.

## Files

- `index.html`: standalone landing page
- `assets/images/`: copied image assets from the live Carrd site
- `CNAME`: GitHub Pages custom domain configuration

## GitHub Pages

1. Push this repo to GitHub.
2. In the repository settings, enable GitHub Pages from the default branch root.
3. Keep the existing DNS pointed at GitHub Pages for `ctrlaltsucceed.com`.
